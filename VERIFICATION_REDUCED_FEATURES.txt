=== MODE 2: FLIGHT PLAN (10 additional features, stacked on Mode 1) ===

Feature Keys (FLIGHT_PLAN_FEATURE_KEYS):
1. gsd_median               <- PRIMARY: Ground Sampling Distance (m/pixel)
2. gsd_missing              <- FLAG: Missing if no altitude/focal/sensor data
3. overlap_proxy            <- PRIMARY: Frame overlap estimate [0-1]
4. overlap_missing          <- FLAG: Missing if no GPS path
5. coverage_spread          <- PRIMARY: Geographic survey extent [0-1]
6. coverage_missing         <- FLAG: Missing if <3 GPS points
7. camera_angle_profile     <- PRIMARY: Gimbal pitch description
8. angle_profile_missing    <- FLAG: Missing if no angle data
9. heading_consistency      <- PRIMARY: Flight path regularity [0-1]
10. heading_missing         <- FLAG: Missing if <3 points or path too short

Concrete Calculations:
✓ gsd_median:           GSD = (altitude × sensor_width) / (focal_length × image_width)
                        Default altitude: 120m, Sensor width: inferred from camera model
                        Output: clamped [0.001, 0.5] m/pixel

✓ overlap_proxy:        avg_step_distance = mean(haversine distances between consecutive GPS points)
                        footprint = 1.2 × altitude
                        overlap = 1.0 - (avg_step / footprint) → clamped [0, 1]

✓ coverage_spread:      lat_span_m = haversine(min_lat, mean_lon → max_lat, mean_lon)
                        lon_span_m = haversine(mean_lat, min_lon → mean_lat, max_lon)
                        bbox_diag = sqrt(lat_span² + lon_span²)
                        spread = bbox_diag / 600.0 → clamped [0, 1]

✓ heading_consistency:  bearings = [bearing between consecutive GPS points]
                        deltas = |turn angles| (shortest arc)
                        turn_std = stdev(deltas)
                        consistency = 1.0 - (turn_std / 90.0) → clamped [0, 1]

✓ camera_angle_profile: if angles: "mean={:.1f},spread={:.1f}" % (mean, stdev)
                        else: "mostly_nadir" | "mostly_oblique"

✓ Missing flags:        0 if GPS/angles present, 1 otherwise

Total Info Content: 5 primary + 5 supporting + 10 from Mode 1 = 20 features combined


=== MODE 3: EXTERNAL IMAGE ANALYSIS (10 additional features) ===

Feature Keys (EXTERNAL_IMAGE_FEATURE_KEYS):
1. vegetation_cover_percentage      <- PRIMARY: Green pixel ratio
2. green_area_missing               <- FLAG: Missing if images unavailable
3. vegetation_complexity_score      <- PRIMARY: Vegetation + texture blend
4. veg_complexity_missing           <- FLAG: Missing if images unavailable
5. terrain_roughness_proxy          <- PRIMARY: Geometric terrain variation
6. roughness_missing                <- FLAG: Missing if no COLMAP/images
7. texture_density                  <- PRIMARY: Edge density measure
8. texture_missing                  <- FLAG: Missing if images unavailable
9. blur_motion_risk                 <- PRIMARY: Image sharpness metric
10. blur_missing                    <- FLAG: Missing if images unavailable

Concrete Calculations:
✓ vegetation_cover_percentage:
  1. For each of 20 processed images:
     - Resize to 128×128 for speed
     - Count pixels where: green > red×1.05 AND green > blue×1.05
     - green_ratio = green_count / total_pixels
  2. Aggregate: vegetation_cover_percentage = median(ratios) → [0, 1]

✓ vegetation_complexity_score:
  - Computed per image: 0.55 × green_cover + 0.45 × texture_density
  - Aggregated: median across 20 images → [0, 1]

✓ terrain_roughness_proxy:
  - PRIMARY METHOD (if COLMAP available):
    1. Read COLMAP sparse points: xyz array from points3D.bin
    2. Bin into 20×20 XY grid (normalize to [0, grid_size))
    3. For each cell with ≥3 points:
       a. Fit plane: z = ax + by + c via least-squares
       b. Compute residuals: |z_i - z_fit|
       c. Cell score = median(residuals)
    4. Final score = median(cell scores) / 10.0 → clamped [0, 1]
  
  - FALLBACK METHOD (if COLMAP unavailable):
    1. Per image: std_luma = std(luma values), roughness = std_luma / 64.0
    2. Aggregate: median across images → [0, 1]

✓ texture_density:
  1. Sobel edges: gx = luma[x+1] - luma[x-1], gy = luma[y+1] - luma[y-1]
  2. edge_sum = sum(|gx| + |gy|) across interior pixels
  3. Normalize: edge_sum / ((W-2)×(H-2)×255) → clamped [0, 1]

✓ blur_motion_risk:
  1. Laplacian: lap = luma[x-1] + luma[x+1] + luma[y-1] + luma[y+1] - 4×luma[x,y]
  2. lap_sum = sum(|lap|) across interior pixels
  3. sharpness = lap_sum / ((W-2)×(H-2)×255) → [0, 1]
  4. blur_risk = 1.0 - sharpness → [0, 1]

✓ Missing flags:        0 if images/COLMAP present, 1 otherwise
